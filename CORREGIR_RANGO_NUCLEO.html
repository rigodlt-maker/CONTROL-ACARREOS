Corregir rango de Núcleo

Sesión: · cerrar sesión

Corrige el campo rango en tickets de material == 'NUCLEO': los que van a destino == 'MURO' deben quedar en 0.02 a 0.20 ton, y los que van a destino == 'ACOPIO MARINO' deben quedar en 0.01 a 0.50 ton. No toca ningún otro campo del ticket. Los tickets pendientes (sin rango capturado todavía) se muestran aparte y NO se tocan.

⚠️ Estás abriendo este archivo directo desde tu celular/computadora (protocolo file://). Esta herramienta usa módulos de JavaScript para conectarse a Firebase y los navegadores los bloquean en ese modo. Súbela a tu repo de GitHub (junto a index.html) y ábrela desde su URL de GitHub Pages.
