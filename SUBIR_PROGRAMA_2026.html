1. Elige el Excel

Debe traer una hoja llamada in, con encabezado en la fila 1 (columna A vacía, columna B "CANTIDAD", y de la columna C en adelante una fecha por columna). Cada bloque empieza con una fila de sección (p. ej. "CUERPO 1", "MORRO", "MURO TARQUINA") seguida de sus filas de material (NUCLEO, CORAZA, etc.) hasta la siguiente sección.

⚠️ Estás abriendo este archivo directo desde tu computadora/celular (protocolo file://). Esta herramienta usa módulos de JavaScript para conectarse a Firebase, y los navegadores los BLOQUEAN en ese modo — por eso "no hace nada", con cualquier archivo que subas. Sube este HTML a tu repo de GitHub (junto a index.html) y ábrelo desde su URL de GitHub Pages, igual que abres la app normal.
